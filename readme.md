## Installation
### Linux 

1. $ mkdir ./build
2. $ cmake -S . -B ./build
3. $ cmake --build ./build 
3. $ ./mandelbrot\_linux 

## Instructions

keyboard       | Description of Action on Fractal
:-------------:|:--------------------------------:
q              | zoom out
e              | zoom in
up             | move upward    
down           | move downward  
left           | move left      
right          | move right
